gulp-playcanvasを使って、
REST API経由でローカルのコードエディタからPlayCanvasにアセットファイルとしてuploadするテンプレプロジェクト。

今まで私が汎用コーディングで使用してきたプロジェクトにgulp-playcanvasを入れただけなのでご注意ください。
